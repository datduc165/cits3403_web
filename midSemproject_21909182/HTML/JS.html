1.JavaScript history and evolution

Alongside HTML and CSS, JavaScript is one of the core technologies of the World Wide Web in order to create a web page. JavaScript is a scripting or programming language that allows you to implement complex behaviors on web page such as dynamic function or update, control multi-media, animate image and many or things.

JavaScript was invented by Brendan Eich in 1995 while in Netscape, and became an ECMA standard in 1997. ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.

  • In November 1996, Netscape submitted JavaScript to ECMA International. Then the official release of the first ECMAScript language specification in June 1997. Then the new release of ECMAScript 2 in June 1998 and ECMAScript 3 in December 1999. ECMAScript 4 was began in 2000 but never released.
  • In 2008, Google introduced Chrome browser, with the V8 JavaScript engine allowing significantly improvement of execution times. This pushed other languages forward in early 2009. In December 2009, the ECMAScript 5 was released .
  • With the ambitious for the better of the language, in 2015, ECMAScript 6 was publiced with an extensive collection of additions.
  • From 2016 to 2019, there was new version each year but with smaller amout of changes.

2.JavaScript statment

Since JavaScript is a programming language, it has instructions to be followed to execute the code. These programming instructions are statments in JavaScript. In order to seperate each statement, a semi-colon is appended at the end of the statment.

Examples of JavaScript statment for declaring variable. 

var a; //Variable declaration with var keywork.
let b; //Variable declaration with let keywork.
const c; //Variable declaration with const keywork.

Note: JavaScript ignore white spaces so that you can add more white space to make your code readable.

2.1 JavaScript Operators

In JavaScript, variables are used to store data values. After declaring a variable, an equal sign (=) is used to assign values to variable or we can declare and assgin value to variable at the same line.

var x;
x = 9;//assgin the value 9 to x.
var a = "text";//declare and assgin "text" to a.

JavaScript also use arithmetic operators (+,-,*,/) to compute values

var x = 1 + 1;//assign the result of the computation to x

With different value type of variable, the operators treat differently.

fname = "Joe";
lname = "Lora";
x = fname + " " +lname;

The result of x will be "Joe Lora". The "+" operator in this case concentrate the strings.

2.2 Data type

Data type of JavaScript variable can be a number, a string, a object and more. 

var length = 16; //Number
var lastName = "Johnson"; // String
var x = {firstName:"Joe", lastName:"Lora"}; // Object

Note: Variables in JavaScript are dynamic which means it a variable can hold different data type.

var x;//x is undefined
x = 5; // x is a number
x = "God"; // x is a string

With data type in JavaScript, it can even compute two different data types.

var a = 1 + "Home";

The result of above code is "1Home" because JavaScript treats number in this case as string. Moreover, JavaScript evalutes from left to right therefore, different sequences have different result.

var a = 1 + 2 +"Home";

After execution, JavaScript will add 1 and 2 and then concentrate it to string "Home" result in "3Home".

3.Add JavaScript

There are several ways to add JavaScript. In HTML, JavaScript is placed inside <script> and </script>. Everything inside it is executed whenever the HTML web page is loaded. JavaScript can be in <head> or <body> section of HTML or in both sections.

Example of JavaScript in the body section.

<body>
    <p id="text_example"></p>
    <button type="button" onclick="changeText()">Let's try</button>
    <script>
        function changeText() 
            {document.getElementById("text_example").innerHTML = "Text changed.";}
    </script>
    </body>

By clicking the button, it loads the JavaScript function(changeText) and change the HTML content of element that has id equal to "text_example".

Note: JavaScript section is recommended to place in the body section before the </body> tag in order to improve the display speed. Moreover, HTML page laods from top to bottom, therefore, some HTML elements won't be available yet to be affected.

Another way to add JavaScript is to add a link to the JavaScript file.

For example, a seperate JavaScript file is used in this project

<script src="../JavaScript/app.js"></script>

This will load all the JavaScript code in the file and manipulate the HTML page. This way allows developers to read and change the code easily.

Beside that, JavaScript file can be loaded from third-site parties using the URL reference.

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/2.1.3/TweenMax.min.js" integrity="sha256-lPE3wjN2a7ABWHbGz7+MKBJaykyzqCbU96BJWjio86U=" crossorigin="anonymous"></script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/2.1.3/TimelineMax.min.js" integrity="sha256-fIkQKQryItPqpaWZbtwG25Jp2p5ujqo/NwJrfqAB+Qk=" crossorigin="anonymous"></script>

In this project two JavaScript files are loaded from a third-site party (CDNJS web page) in order to add the animation for the banner of this project. After empployed, all the function in those JavaScript files can be declared and used in our main JavaScript file.

4. JavaScript & HTML DOM

HTML DOM stands for Document Object Model of the page. It represents the web page so that it can be accessed and manipulated by programming language such as JavaScript.

4.1 DOM

JavaScript can get all the HTML elements or attributes or CSS selector using method.

Recall the example above.

<body>
    <p id="text_example"></p>
    <button type="button" onclick="changeText()">Let's try</button>
    <script>
        function changeText() 
            {document.getElementById("text_example").innerHTML = "Text changed.";}
    </script>
</body>

A function changeText is declared and assigned as an event whenever the button is clicked.document.getElementById is the method which get a HTML element that have id = "text_example" and innerHTML is called the property of that method which can change or set the content of the element.

There are several ways to find HTML elements or CSS selector

  • document.getElementById(id) - get element by id
  • document.getElementsByTagName(name) - get elements by HTML tag name
  • document.getElementsByClassName(name) - get elements by class name
  • document.querySelector(name) - get a CSS selector whether by HTML element name or by class name(.class) or by id name(#id)

The follow JavaScript code is used in this project to make the effect of the above banner.

const ban = document.querySelector(".banner");
const tl = new TimelineMax();
tl.fromTo(ban,1.5,{width:"0%"},{width:"100%"}

The first variable is declared and assigned with the return result of document.querySelector(".banner"). If there is no selector that have a class name of "banner", a null value is stored in variable ban. Meanwhile, the second variable is declared to store a new function of TimelineMax() which is employed from a third-site party (CDNJS web page).

The tl variable is then used to set the animation of the banner which is stored in ban variable. This function of tl allows the banner of this project to change its width from 0% to 100% in 1.5 second.

Note: querySelector and getElementById only return the first result in case there are more than one result. Other methods return a collection of results. Navigation between HTML elements is explained here.

Beside changing the content of HTML elements with innerHTML, we can change the attribute or style of the elements.

  • element.attributeattribute = new value OR element.setAttribute(attribute, value) - Change the attribute value of an HTML element
  • element.style.property = new style - Change the style of an HTML element

The following is a simple feature that allows the user to change the background color using what was stated above.

<!--HTML part-->
    <input id="BGcolor">
    <button class="button" type="button" onclick="ChangeBGColor()">Change</button>
    <button class="button" type="button" onclick="ResetBGColor()">Reset</button>
<!--HTML part-->
/*JavaScript part*/
function ChangeBGColor() {
    var x;
    x = document.getElementById("BGcolor").value;
    document.querySelector("body").style.backgroundColor = x;
  } 
  function ResetBGColor() {
    document.querySelector("body").style.backgroundColor = "";
  }

The first JavaScript function gets the input color value with getElementById("BGcolor")> and assgin its value to variable x using the property .value. Then, we find the body selector in CSS using document.querySelector("body") and set its background color with the input color (document.querySelector("body").style.backgroundColor = x).

4.2 DOM Navigation

HTML DOM Tree

With HTML DOM, everythins in HTML page is known as a node that follows parent-relationship. The above image demonstrates a relationship tree of HTML elements and attributes which can be access by JavaScript.

From the above image, <html> is known as a root node that is the parent of <head> and <body>. Therefore, <head> is known as the first child and <body> is the last child of <html>

Using parent-child relationship, JavaScript can navigate to get the right node.

<script>
    const y = document.getElementById("source");
    document.getElementById("here").innerHTML = y.firstChild.nodeValue;
</script>

Using the above JavaScript, it will get the nodeValue of HTML element that has id="source" and then assigns it to HTML element that has id="here". The result is the two same texts.

5. JavaScript AJAX

AJAX stands for Asynchronous JavaScript And XML. It is a techniqure that allows web page to load new data from a web server without reloading a whole page. In AJAX, XMLHttpRequest object is the keystone. The following example will explain how AJAX work in order to get data from web server.

/*JavaScript-AJAX part*/
    function load(url, cFunction) {
    var xhttp;
    xhttp=new XMLHttpRequest();
    xhttp.onreadystatechange = function() {
        if (this.readyState == 4 && this.status == 200) {
        cFunction(this);
        }
    };
    xhttp.open("GET", url, true);
    xhttp.send();
    }
    function change(xhttp) {
    document.getElementById("ajax_text").innerHTML = xhttp.responseText;
    }
 /*HTML part*/   
    <p id="ajax_text"></p>
    <button class="button" onclick="load('ajax_text.txt', change)">Click</button>

XMLHttpRequest() is declared to exchange data with a server. Then using the onreadystatechange Property to know when a function need to be executed when the readyState changes. The status property holds the status of the XMLHttpRequest object to know whether the page is "200-OK" or "400-Page not found and so on.

xhttp.open("GET", url, true) specifies a "GEt" request to get the data using the given "url". The "true" value is a boolean parameter which defines whether or not to perform the operation asynchrinously. With "true" value, xhttp.send() method returns as soon as the request is sent.

After receiving the result, change() find the element with id ="ajax_text" to assgin the result.

Note: the url, "ajax_text.txt", is the placed inside the same folder for this example.
