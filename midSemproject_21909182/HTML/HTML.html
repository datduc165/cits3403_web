1. Hypertext Markup Language(HTML) history and evolution

In the late 1991, Sir Tim Berners-Lee created his first elementary browsing and authoring system for the Web which was a bunch of baisc hypertext language (HTML) but he did not released officially and then in 1995, this language was published officially as HTML 2.0 and then other version of HTML have been updated to imporve the efficiency of making web page.

HTML is a standard markup language which is a set of Markup tags used for document to be represented in a website and has been evolved with various versions to improve the features of HTML. The implemented features were decided by the Web community over the years in order to allow a much easier and more efficiency way to make a web site.

  • HTML 1.0 was released in 1993 with the intention of sharing information which can be readable and accessible via web browsers. But not much of the developers were involved in creating websites.
  • Then comes the HTML 2.0, published in 1995; which contains all the features of HTML 1.0 along with that few additional features; which remained as the standard markup language for designing and creating websites until January 1997 and refined various core features of HTML.
  • Then comes the HTML 3.0, where Dave Raggett who introduced a fresh paper or draft on HTML. It included improved new features of HTML, giving more powerful characteristics for webmasters in designing web pages. But these powerful features of new HTML slowed down the browser in applying further improvements.
  • Then comes the HTML 4.01 which is widely used and was a successful version of HTML before HTML 5.0, which is currently released and used worldwide. HTML 5 can be said for as an extended version of HTML 4.01 which was published in the year 2012.

2.HTML tag & element

HTML tags can be considered as commands incorporated in HTML, which holds the capability to define how a browser will display the content and format of the web page. They are enclosed in angle brackets.

For example: All HTML documents begin with <!DOCTYPE html> which is inside angle brackets, informs the web browser about the document type.

Most HTML elements have two sections: an opening and a closing tag. Moreover,the context of a HTML element which is between the starting tag and ending tag is defined to display in a web page.

The <html>element have <html> tag as the starting tag and </html> as the ending tag which tells the browser that this is an HTML document and it is the container for all other HTML elements (except for the <!DOCTYPE> tag).

There are also HTML elements that only have one section (unpaired tag) such as <!DOCTYPE html> , <br>(create a line break in web page without having any content or text) or <img>(embed an image with a reference to the image resource). 

<img src="../CSS/theme_2.jpg" alt="theme">

<img> tag in this page is used in order to define an image from a file with the given URL.

HTML elements can be nested which means elements can be inside other elements before the ending of that element.

In HTML,there are other tags such as <head>, <body> or <p> and they are all nested inside <html> tag 

<html>
    <body>
        <p>Result: like what were build in this <b>website</b> </p>
    </body> 
</html>

Result: like what were build in this website

The tag <body> which difines a page content is placed inside <html>

The <b> tag is nested inside a <p> tag to make the content "website" become bold by the <b> element.

In HTML, there are many tags for web developers to explore and make their website efficiency. In this project, a navigation tag (<nav>) and a unordered list tag (<ul>) are used in order to make a navigation bar at the top of this web project.

<nav>
    <ul>
        <li><a href="HTML.html">HTML</a></li>
        <li><a href="CSS.html">CSS</a></li>
        <li><a href="JavaScript.html">JavaScript</a></li>
        <li><a href="About.html">About</a></li>
    </ul>
</nav>

A (<nav>) block is used to contain an unordered list (<ul>) of URL links which is included by anchor tag (<a>) so that the web page will render "HTML" as a hyperlink. With appropriate CSS (explained in this link),this is presented as a navigation bar to other web pages.

3. HTML attribute

Most of the attributes of an element are name-value pairs, separated by = and written within the start tag of an element after the element's name. The value may be enclosed in single or double quotes. Attributes provide additional information about HTML elements.

Given the image tag example above:

<img src="../CSS/theme_2.jpg" alt="theme">

"src" attribute in this tag provide the resource of image and "alt" attribute define an alternative name of the image whenever the image is unavailabe.

Or in the narvigation bar above.

<li><a href="HTML.html">HTML</a></li>

href attribute provides the link target (HTML.html) and then the content is render as a hyperlink to that target link.

Note: Those links above is known as relative links

HTML attribute can be use to provide some style with "style" attribute in HTML which determine some CSS for a element. The syntax of style attribute follow CSS syntax.

With this following HTML code:
<p style = "color:maroon;font-size:25px" > Result: This text is style with style attribute</p>

Result: This text is style with style attribute

The style attribute above is declared to make the content of that specific <p> element has maroon color for the text and a font-size of 25px.

Beside of above attributes there is also class and id attribute which give name to one or more HTML elements. With this attribute, CSS and JavaScript can perform their tasks on specific elements using class name( more on CSS&JavaScript).

<div class="example">...<div>

All the example boxes in this project is defined a class name of "example". And this "example" element is style in CSS file to have a blue-purple rectangle for this web project.

4. Font Formatting

Despite the style attribute in HTML, font in HTML can be formatted by HMTL elements.Formatting elements were designed to display special types of text:

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Small text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

By adding <b> tag and its content will become bolded for example.

<p>This text is <b>bolded</b>.</p>

This text is bolded.

Or use <i> to make the text italicized.

<p>This text is <i>italicized</i>.</p>

This text is italicized.
